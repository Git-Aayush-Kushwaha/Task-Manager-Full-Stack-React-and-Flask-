# Full-Stack Task Manager

A beautiful, modern Task Manager built with a Python (Flask) backend and a React frontend, featuring a custom glassmorphism UI.

## Getting Started

### 1. Start the Backend
Navigate to the root directory, activate your virtual environment, and run:
\`\`\`bash
pip install -r requirements.txt
python app.py
\`\`\`

### 2. Start the Frontend
Open a new terminal, navigate to the frontend directory, and run:
\`\`\`bash
cd frontend
npm install
npm start
\`\`\`

The app will run automatically on `http://localhost:3000`.
